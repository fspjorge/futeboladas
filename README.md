# 🏟️ Futeboladas

Aplicação móvel Flutter para gestão e marcação de jogos de futebol entre amigos.  
Desenvolvida com **Firebase** e **Flutter**, oferece autenticação via **Google** e **email**, e uma interface moderna inspirada em tons de **relva** e **ambiente urbano**.

---

## 📱 Funcionalidades Atuais

- 🔐 Autenticação com **Google**  
- ✉️ Autenticação com **email e palavra-passe**  
- ✅ Validação de email após registo  
- 🔄 Login silencioso (mantém sessão ativa)  
- 🚪 Logout e remoção de conta  
- 🧾 Atualização de dados do utilizador (nome, palavra-passe, etc.)

---

## 🛠️ Tecnologias Utilizadas

| Tecnologia | Função |
|-------------|--------|
| **Flutter 3.35.7** | Framework principal |
| **Dart 3.9.2** | Linguagem de programação |
| **Firebase Auth** | Autenticação (Google e email) |
| **Firebase Core** | Integração base com Firebase |
| **Google Sign-In** | Login Google no Android e Web |
